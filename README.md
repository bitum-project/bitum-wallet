# Bitum Blockchain - Offline Wallet

Clone project
```
  git clone git@github.com:bitum-project/bitum-wallet.git
  cd bitum-wallet/
```

Install dependencies
```
  npm i

  # After installation you need run patch
  node patch.js
```

Run with dev env
```
  npm start

  # open browser => http://localhost:4200
```
